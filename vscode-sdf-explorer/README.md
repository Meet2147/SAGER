# SDF Explorer

VS Code extension for visualizing `.sdf` files generated by SAGER as a tree structure.

## Features

- open any `.sdf` file
- inspect top-level metadata
- browse atoms grouped by page
- browse edges grouped by type
- inspect annotations and statistics

## How to use

1. Open this extension folder in VS Code.
2. Do not run `node extension.js` directly.
3. Press `F5` to launch the extension host.
3. In the new VS Code window, open an `.sdf` file or use `SDF Explorer: Open SDF File`.
4. Browse the tree in the `SDF` activity-bar view.
