# Overleaf Bundle

Upload the contents of this folder directly into a new Overleaf project.

## Main file

- `paper.tex`

## Included assets

- `references.bib`
- `figures/*.png`

## After upload

1. Set the main file to `paper.tex` if Overleaf does not detect it automatically.
2. Update the author email and affiliation block.
3. Review the acknowledgment and AI-use disclosure for the target venue.
4. Compile with `pdfLaTeX` + `BibTeX` or Overleaf's default LaTeX build.

## Important

Before submission to IEEE:

- choose a specific venue
- confirm page limits
- confirm AI disclosure policy
- replace any placeholder metadata
