## Practical note

IEEE does not accept "publish to IEEE" directly as a single action. We must:

1. pick a specific IEEE journal or conference
2. adapt the paper to that venue's scope and page limits
3. prepare final references, authorship, and disclosures
4. submit through that venue's official system

## Important disclosure item

Because AI assistance was used during drafting, review the target venue's policy and include any required disclosure. For example, current IEEE Access author guidance says AI-generated text, figures, images, or code used in a submission should be disclosed in the acknowledgments section, with the system identified and the specific sections described.
