# VS Code Extension for SDF

## What it is

A lightweight VS Code extension that visualizes `.sdf` files as a tree structure.

Folder:

`vscode-sdf-explorer/`

## What it shows

- file metadata
- method metadata
- source information
- stats
- atoms grouped by page
- edges grouped by type
- annotations

## How to run it

1. Open `vscode-sdf-explorer/` in VS Code.
2. Press `F5`.
3. In the extension host window, open an `.sdf` file.
4. Use the `SDF` sidebar view.
